FAQs

Punya pertanyaan ?

MAPPA ID menjawab pertanyaan yang sering ditanyakan

  • Bagaimana caranya update metadata dan atau hapus MBTILES?

    Klik icon pencarian(icon kaca pembesar), masukkan nomor grid kemudian klik Search. Bila email di metadata sama dengan email yang digunakan untuk login maka akan ada link untuk Edit dan Download.

  • Apa manfaatnya upload MBTILES ke mappa?

    Sebagai kontributor, minimal punya portofolio kemampuan ngedron yang hasilnya bisa dipresentasikan/dilihat secara online tanpa harus investasi server dan aplikasi penunjang.

  • Kenapa penampilan di pingir AOI putih/hitam?

    Pelajari cara menbuat MBTILES di link berikut https://mappa.id/Cara_membuat_Tile_Map_MBTILES.pdf

  • Kenapa di Global Mapper nomor grid tidak muncul?

    Sekalipun file *.mif bisa tampil namun datanya ada di file *.mid , jadi harus didownload dua duanya baru file *.mif nya dibuka di Global Mapper
